# Three Game — версия 2

Игровой проект на Flutter Web. В этом репозитории сохранена версия 2 с чистой историей git.

**Ссылка на GitHub Pages**: https://s7018025085.github.io/Three_game_v2/

**Быстрый старт (локально)**
- **Требования:** установлен Flutter SDK (рекомендуется Flutter 3+), `git`.
- Откройте терминал в папке проекта:
  ```powershell
  cd C:\PROJECT\FLUTTER\Three_game
  ```
- Запустить в режиме разработки (быстро проверять в браузере):
  ```powershell
  flutter run -d web-server --web-hostname 0.0.0.0 --web-port 8080
  ```
  После запуска откройте в браузере: `http://localhost:8080`.
- Построить релизную версию для веба (генерирует папку `build/web`):
  ```powershell
  flutter build web --release
  ```
  Содержимое `build/web` можно развернуть на любом статическом хостинге (GitHub Pages, Netlify и т.д.).

**Публикация на GitHub Pages**
- Репозиторий: `s7018025085/Three_game_v2`.
- Веб-страница доступна по адресу: `https://s7018025085.github.io/Three_game_v2/`.
- Если вы делаете новый билд и хотите обновить Pages:
  1. Скопируйте содержимое `build/web` в корень репозитория (или настроите CI для деплоя).
  2. Закоммитьте и запушьте в ветку `main`.
  3. Перейдите в `Settings → Pages` и убедитесь, что Source = `main` и Folder = `/ (root)`.

**Инструкции для iPhone / PWA**
- Откройте страницу в Safari: `https://s7018025085.github.io/Three_game_v2/`.
- Чтобы добавить как приложение на главный экран: нажмите кнопку «Поделиться» → «Add to Home Screen».
- При проблемах с загрузкой (старые версии, service worker):
  - Откройте Safari → нажмите кнопку обновления (pull-to-refresh) или откройте DevTools (через Mac) и отключите service worker.
  - В крайнем случае очистите кеш сайта в настройках Safari.
